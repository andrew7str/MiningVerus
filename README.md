# How To Use Via Linux

1. sudo apt update && sudo apt upgrade -y
2. sudo apt install git build-essential cmake libssl-dev libgmp-dev -y
3. git clone https://github.com/andrew7str/MiningVerus/
4. cd MiningVerus
5. mkdir build && cd build
6. cmake ..
7. make -j$(nproc)
